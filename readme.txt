== Description ==

Displays server environment details in the WordPress admin dashboard.

== Installation ==

1. Upload `ae-system-info` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How do I view the system info? =

After activating the plugin, go to `System Info` in the WordPress admin menu.
